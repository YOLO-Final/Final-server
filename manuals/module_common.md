# COMMON 모듈 운영 가이드

## 1. 한 줄 설명

여러 모듈에서 공통으로 쓰는 RBAC/Audit 보조 유틸을 제공합니다.

## 2. 주요 구성

- `rbac.py`: 역할 값 검증용 placeholder
- `audit.py`: 감사 로그 구조 placeholder

## 3. 운영 관점 설명

- 현재는 본격 권한 체계의 완성본이 아니라 기본 뼈대 성격입니다.
- 실제 인증/권한 핵심은 `auth` 모듈과 JWT 로직이 담당합니다.

## 4. 점검 포인트

1. 권한 정책 변경 작업 시 common + auth를 함께 검토
2. 감사로그를 DB에 남기려면 audit placeholder 확장이 필요

## 5. 자주 발생하는 오해

- 오해: common만 수정하면 권한 정책이 모두 바뀐다
- 실제: auth/JWT/라우터 의존관계까지 함께 반영해야 적용됨
