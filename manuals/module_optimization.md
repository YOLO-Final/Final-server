# OPTIMIZATION 모듈 운영 가이드

## 1. 한 줄 설명

최적화 기능의 상태 점검용 모듈입니다.

## 2. API 기본 경로

- `/api/v1/optimization`

## 3. 주요 기능

- 최적화 모듈 상태 확인

## 4. 주요 엔드포인트

- `GET /api/v1/optimization/status`

## 5. 운영 점검 포인트

1. 상태 응답이 정상인지 주기 점검
2. 실제 최적화 로직 연계 시 별도 서비스 로그와 함께 확인

## 6. 자주 발생하는 문제

- 증상: 모듈 연결만 확인하고 싶음
- 조치: status 엔드포인트로 생존 확인
